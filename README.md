## Features
1. Reusable task dialog window builder (with preview)
2. Audio and video (renders on desktop) support
3. Action timestamps
    - Show window
    - Hide/Show windows
    - Close windows
    - Change window positions
    - Animate window positions
    - more to come...
4. Bundle configs into single file executable
